# eMMC 工作模式

## Overview

系统 Power On、HW Reset 或者 SW Reset 时，如果 Host 有触发 Boot From eMMC Device 流程，那么 eMMC Device 进入 Boot Mode。如果没有触发 Boot 流程，那么就会跳过 Boot Mode 直接进入 Device Identification Mode。
